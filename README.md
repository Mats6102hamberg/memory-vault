# Memory Vault

Detta repo är Mats driftcentral för alla appar, agenter och experiment. Här lagras beslut, loggar och handover så att arbetet kan återupptas även efter 18 månader utan sammanhang.

## Syfte
- Vara permanent minnesbank för projekt och agenter
- Säkerställa att samma kunskap finns för produkt, teknik och design
- Ge snabb väg in i projekten via NOW/HANDOVER/DECISIONS

## Så används repo:t
1. Börja alltid i `start-here/START.md` för översikt
2. Följ arbetsflödet i `start-here/HOW_TO_USE_THIS.md`
3. Uppdatera respektive projektmappar efter varje arbetspass
4. Lägg till loggar i `logs/CHANGELOG.md`
5. Deploya frontend under `frontend/` till Vercel för publik visning

## Driftcentralen
Memory Vault fungerar som Mats nav:
- Vision och produktbeslut dokumenteras här
- Teknikstrategi och checklistor hålls synkade
- Handover för externa utvecklare pekar hit

Inga hemligheter eller API-nycklar ska någonsin lagras i detta repo.
